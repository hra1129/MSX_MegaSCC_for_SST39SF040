SST39SF040 を使った MegaSCC の作り方とツール一式
How to build MegaSCC with SST39SF040 and a set of tools
-------------------------------------------------------------------------------
1. Introduction
	MSX用のフラッシュROMカートリッジである MegaSCC の作り方です。
	AM29F040B を使った MegaSCC の作り方は、いろいろなところで公開されており、
	OPF.COM 等の書き込みツールが公開されています。
	しかし、2022年5月現在 AM29F040B は、すでに入手困難になっており、偽物も
	出回っている状況です。
	そこで、AM29F040B と比べれば入手しやすい SST39SF040 を使って MegaSCC を
	作れないか？と対策を講じたので、ここにまとめておきます。

	This is how to make MegaSCC, a flash ROM cartridge for MSX.
	How to make a MegaSCC using AM29F040B is published in various places. 
	OPF.COM and other writing tools are available.
	However, as of May 2022, AM29F040B is already difficult to obtain, 
	and counterfeit copies and counterfeit copies are also circulating.
	Therefore, I have devised a solution to create MegaSCC using SST39SF040, 
	which is easier to obtain compared to AM29F040B, and I will summarize 
	it here.
